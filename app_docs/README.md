# Application Documentation

Documentacion de tac-bootstrap.

## Estructura

```
app_docs/
├── features/          # Documentación de features específicas
├── api/              # Documentación de API endpoints
└── guides/           # Guías de uso y desarrollo
```

## Generacion

La documentación se puede generar usando el comando:

```bash
claude -p "/document [feature]"
```

Este directorio contiene documentación técnica sobre la aplicación, incluyendo:
- Arquitectura y diseño
- APIs y endpoints
- Guías de desarrollo
- Tutoriales y ejemplos
