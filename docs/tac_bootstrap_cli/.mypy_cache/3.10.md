# Overview

Mypy cache folder for tac-bootstrap CLI.

# Responsibilities

* Cache mypy type checking results.
* Enable faster development and testing.

# Key APIs / Components

* `mypy_cache`
* `typing_inspection`

# Invariants & Contracts

* Cache is updated on every import.
* Type checking results are cached for a short period of time.

# Side Effects & IO

* Reads/writes to disk.
* Uses system resources.

# Operational Notes (perf, scaling, failure)

* Can be slow due to disk I/O.
* May fail if disk space is low.
