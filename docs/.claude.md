# .claude

Claude Code configuration and command definitions for TAC Bootstrap.

## Overview

The `.claude/` directory contains Claude Code settings, command definitions, and automation hooks. This is the configuration layer for AI-assisted development workflows integrated with the TAC Bootstrap project.

## Contents

- **settings.json** - Claude Code permissions, hooks, and settings
- **commands/** - Slash command definitions (25+ commands)
- **hooks/** - Automation scripts triggered by events

## Key Files

### conditional_docs.md
Conditional documentation guide for command generation. This file helps structure documentation that adapts based on context and conditions, enabling smart documentation generation based on project state.

## Architecture

The `.claude/` configuration enables:
- Custom slash commands for development workflows
- Event hooks for automation (pre-commit, post-generate, etc.)
- Permission management for agent operations
- Settings for Claude Code behavior

## Command Categories

- **Development**: `/prime`, `/scout`, `/feature`, `/implement`, `/test`, `/review`, `/commit`
- **Generation**: `/generate_fractal_docs`, etc.
- **ADW Workflows**: Triggers for SDLC, patching, and issue tracking

## Related Directories

- `adws/` - AI Developer Workflows that are configured and triggered via hooks
- `scripts/` - Utility scripts executed by hooks
- `docs/` - Generated documentation for the project
